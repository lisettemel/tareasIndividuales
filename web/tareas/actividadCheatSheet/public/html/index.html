CSS Cheat Sheet

Border

The border property sets the border around an element. It is a shorthand for setting border-width, border-style, and border-color.

This is a box with a border.

Border Left, Border Right, Border Top, Border Bottom

These properties set the individual borders on each side of an element.

This box has individual borders.

Border Radius

The border-radius property sets the rounded corners of an element's outer border edge.

This box has rounded corners.

Background Image

The background-image property sets an image as the background of an element.

This box has a background image.

Color

The color property sets the color of the text inside an element.

Font

The font property is a shorthand for setting various font properties in one declaration.

Font Family

The font-family property specifies the font for an element's text.

Font Style, Font Weight

The font-style property specifies whether the text is italic, and the font-weight property specifies the weight (or boldness) of the font.
